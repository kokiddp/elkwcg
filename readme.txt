== Description ==

This plugin adds some useful functions to WooCommerce

Configuration options are available in
WooCommerce->Settings->Products->ELK WooCommerce Goodies

== Installation ==
 
1. Install from Plugin page or extract 'elkwcg.zip' in the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure it in WooCommerce->Settings->Products->ELK WooCommerce Goodies
